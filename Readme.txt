Blackboard Download Manager Ver 1.6 - MT
******************************************

1. Copiar la carpeta "BckLib" en C:\\ <--Librerias y excel con enlaces
2. Copiar la carpeta "BlackboardDownloadManager" en C:\\ <--- Ejecutable
3. Una vez copiadas las carpetas en C://
4. Configurar el archivo Excel C:\BckLib\Archives\Links.xlsx y dejar sin formato.
5. Ya no hace falta borrar el contenido de los ".txt" de Enlaces y nombres, el solo lo hace al ejecutarse.
6. Abrir "CMD" y copiar en la consola: java -jar "C:\BlackboardDownloadManager\BlackboardExtractVideosTest.jar" 
7. Se verá la nueva versión.

******************
Atención: No cerrar la ventana de las descargas hasta que todas se hayan terminado, de lo contrario se cerrará por completo.
